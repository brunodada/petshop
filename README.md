# 🐾 Pet Shop Landing Page

Landing page institucional moderna desenvolvida para apresentação de um
**Pet Shop**, com foco em transmitir confiança aos clientes e facilitar
o contato para serviços relacionados ao cuidado de animais de estimação.

O projeto possui layout limpo, responsivo e otimizado para navegação em
dispositivos móveis, proporcionando uma experiência agradável e
intuitiva para os usuários.

------------------------------------------------------------------------

## 📌 Objetivo do projeto

Criar uma página institucional simples, moderna e funcional para
pequenos negócios locais, permitindo apresentar serviços, informações da
empresa e facilitar o contato com clientes.

A página pode ser facilmente adaptada para outros tipos de negócios.

------------------------------------------------------------------------

## ✨ Funcionalidades

-   Layout moderno e profissional
-   Design responsivo (mobile-first)
-   Seção de apresentação do pet shop
-   Exibição dos serviços oferecidos
-   Seção de diferenciais
-   Depoimentos de clientes
-   Galeria de imagens
-   Formulário de contato
-   Botão de contato via WhatsApp
-   Integração com Google Maps
-   Navegação suave entre seções

------------------------------------------------------------------------

## 🧑‍💻 Tecnologias utilizadas

-   JavaScript
-   Design Responsivo
